# goutils

Go Utils

常用功能放在goutils包
大的模块功能放在子包内

```shell
go mod tidy
go mod download

go get -u github.com/BurntSushi/toml
go get -u go.mongodb.org/mongo-driver

go list -m -json all
```

```shell
git tag -ln
git tag -a v0.0.7 -m "goutils-v0.0.7"
git push origin --tags
```
